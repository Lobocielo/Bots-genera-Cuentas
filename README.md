# ZENIHT Account Generator ✦

Bot premium para generación de cuentas de Free Fire, hospedado 24/7.

## Estado
El bot se ejecuta automáticamente vía GitHub Actions.

## Comandos
- `/crear`: Genera cuentas en la región especificada.
- `/info`: Muestra tu perfil y días de suscripción restantes.
- `/add_client`: (Dueño) Añade suscriptores.
- `/list_clients`: (Dueño) Lista clientes activos.
- `/broadcast`: (Dueño) Mensaje global.
